# freeCodeCamp Relational Database Certification Portfolio

This repository contains a collection of five core projects completed for the **freeCodeCamp Relational Database Certification**. These projects represent a comprehensive journey through database design, SQL implementation, Bash automation, and data normalization.

## 🎯 Certification Objectives

The primary goal of this curriculum was to master the **PostgreSQL** ecosystem. Throughout these five projects, I developed technical proficiency in:

* **Relational Schema Design:** Designing databases from scratch based on complex user stories.
* **SQL Mastery:** Advanced use of `SELECT`, `JOIN`, `INSERT`, `UPDATE`, and `DELETE` commands.
* **Automation:** Writing Bash scripts to automate data ingestion and database management.
* **Data Integrity:** Implementing `PRIMARY KEY`, `FOREIGN KEY`, `UNIQUE`, and `NOT NULL` constraints.
* **Version Control:** Managing database states using `pg_dump` and Git for a professional workflow.

---

## 📂 The Five Core Projects

### 1. [Celestial Bodies Database](https://www.google.com/search?q=./celestial-bodies-db/)

* **Focus:** Basic Schema Design & SQL Syntax.
* **Description:** Built a 5-table database (Galaxy, Star, Planet, Moon, Type) with 60+ rows of data.
* **Skills:** Data types (`INT`, `NUMERIC`, `TEXT`, `BOOLEAN`), Primary Keys, and Foreign Keys.

### 2. [World Cup Database](https://www.google.com/search?q=./world-cup-db/)

* **Focus:** Data Normalization & Bash Automation.
* **Description:** Created a script to parse a `games.csv` file and insert match data into a normalized schema without duplicates.
* **Skills:** Bash Scripting, `WHILE` loops, and conditional SQL logic.

### 3. [Salon Appointment Scheduler](https://www.google.com/search?q=./salon-scheduler/)

* **Focus:** Interactive Bash Programs.
* **Description:** Developed a script that allows users to book appointments by interacting with a PostgreSQL backend.
* **Skills:** User input handling, SQL subqueries, and formatted terminal output.

### 4. [Periodic Table Database](https://www.google.com/search?q=./periodic-table/)

* **Focus:** Database Refactoring & Cleaning.
* **Description:** Took a "messy" database and refactored it to meet strict naming conventions and data standards.
* **Skills:** `ALTER TABLE`, `RENAME COLUMN`, and handling `NULL` values.

### 5. [Number Guessing Game](https://www.google.com/search?q=./number-guess/)

* **Focus:** Database State Management.
* **Description:** A Bash-based game that tracks user history, games played, and "best scores" in a relational database.
* **Skills:** Tracking session data and updating existing records based on user performance.
